# Heroes App

## Levantar Desarrollo


1. Clonar el proyecto
2. Editar el archivo `.env` con las variables de entorno basadas en el archivo `.env.template`
3. Ejecutar  `npm install`
4. Ejecutar  `npm run dev`
5. Veriifcar que el backend este levantado
6. Abrir el navegador en `http://localhost:5173`
